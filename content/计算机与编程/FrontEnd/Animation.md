# 前端动画库

GreenSock's GSAP JavaScript animation library (including Draggable). 

- [GitHub](https://github.com/greensock/GSAP)

## POSE

A truly simple animation library for React, React Native, and Vue.

- [Website](https://popmotion.io/pose/)
